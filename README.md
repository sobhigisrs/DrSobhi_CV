# Dr. Sobhi Abdel Hameed Abdel Gawad Geospatial Portfolio

Premium static portfolio website for an academic researcher, Assistant Professor, PMP certified senior GIS and Remote Sensing consultant, GeoAI researcher, and applied environmental risk specialist.

Repository: https://github.com/sobhigisrs/DrSobhi_CV.git

GitHub Pages URL: https://sobhigisrs.github.io/DrSobhi_CV/

## Features

- Responsive GitHub Pages-ready HTML/CSS/JavaScript project
- Bootstrap 5, AOS, Typed.js, Swiper.js, Font Awesome, and Leaflet.js
- Executive academic and consulting copy rewritten from the CV
- Interactive GIS dashboard with countries, projects, and research locations
- Individual project case-study pages
- Searchable and filterable publications with citation and BibTeX copy buttons
- Consulting, skills, academic profile, gallery, downloads, contact, and print-friendly CV pages
- PWA manifest, service worker, offline cache, 404 page, sitemap, robots file, and SEO metadata

## GitHub Pages Deployment

1. Push this folder to `https://github.com/sobhigisrs/DrSobhi_CV.git`.
2. Open repository settings.
3. Enable GitHub Pages from `main` branch and `/root`.
4. Keep `.nojekyll` in the root so all static assets are served directly.

## Recommended Git Commands

```bash
git init
git branch -M main
git remote add origin https://github.com/sobhigisrs/DrSobhi_CV.git
git add .
git commit -m "Build premium geospatial portfolio"
git push -u origin main
```

If the repository already contains files, use:

```bash
git pull origin main --allow-unrelated-histories
git push -u origin main
```
